# AURA - Artificial Unified Responsive Assistant 🤖

AURA is a modern web-based virtual voice assistant built using HTML, CSS, and JavaScript. It can interact with users through voice commands, perform basic tasks, and provide an intelligent assistant-like experience directly in the browser.

## 🌐 Live Demo
🔗 https://aura-assistant-ai.netlify.app

## 📌 Features

- 🎤 Voice recognition using Web Speech API  
- 🧠 Responds to user voice commands  
- 🌐 Opens websites like Google, YouTube, etc.  
- ⏰ Tells current time and date  
- 💬 Interactive and responsive UI  
- 🎨 Modern futuristic interface design  
- ⚡ Fast and lightweight web-based assistant  

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- Web Speech API  
- Font Awesome  

## 🚀 How to Run Locally

1. Clone the repository:

git clone https://github.com/Hibha-2004/AURA-Virtual-Assistant.git


2. Open the folder:

cd AURA-Virtual-Assistant


3. Open `index.html` in your browser

## 🎯 Example Commands
